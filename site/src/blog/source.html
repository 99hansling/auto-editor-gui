Why Canâ€™t Source Files Be Added Directly?

File paths can have almost any character, including , =, which is a problem because auto-editor uses those characters for syntax.

auto-editor movie.mp4 movie2.mp4 --source funydog:/Users/wyattblue/Downloads/dog-123.png \
--add image:0,30,src=funydog

funydog is the label that points to the file path.

sources:
  0 -> movie.mp4
  1 -> movie2.mp4
  funydog -> /Users/wyattblue/Downloads/dog-123.png

Users defined labels cannot:

  • Be greater than 500 characters
  • Start with a digit or a dash
  • Contain , = . : ; ( ) / \ [ ] { }' " | # < > & ^ % $ _ @ anywhere
  • Be an invalid UTF-8 character
  • Contain the space character
